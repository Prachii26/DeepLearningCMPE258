#### Setup & Installation

**Prerequisites:**
```bash

```

**Running the App:**
```bash
# Navigate to project directory

# Install dependencies

# Run on connected device/emulator

# Build APK for Android
flutter build apk --release

# Build for iOS (requires macOS)
flutter build ios --release
```

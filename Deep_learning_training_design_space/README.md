# 🧠 Deep Learning Training Design Space

This repository contains a collection of **hands-on deep learning tutorials** implemented in Jupyter Notebooks. It is designed to help learners understand core concepts through practical implementation along with guided video explanations.

---

## 📂 Repository Structure

```
Deep_learning_training_design_space/
│
├── Video/
│   └── VideoURL.txt        # YouTube links explaining each notebook
│
├── Final_activation_functions_tutorial(Prachi_Gupta).ipynb
├── Final_cnn_fundamentals_tutorial(Prachi_Gupta).ipynb
```

## 📘 Tutorials Included

### 🔹 Activation Functions
- Sigmoid, ReLU, Tanh, Softmax
- Advantages & limitations
- Visualization and implementation

### 🔹 CNN Fundamentals
- Convolution, pooling, feature maps
- CNN architecture
- Practical implementation

### 🔹 Hyperparameter Tuning
- Learning rate, batch size, epochs
- Grid search and tuning strategies
- Model performance improvement

### 🔹 Classification Metrics
- Accuracy, Precision, Recall, F1-score
- Confusion Matrix
- Model evaluation techniques

### 🔹 Optimizers in Deep Learning
- Gradient Descent variants
- Adam, RMSprop, SGD
- Optimization comparisons

## 🚀 Getting Started

### ✅ Prerequisites
- Python 3.x
- Jupyter Notebook / Google Colab
- Required libraries:

```bash
pip install numpy matplotlib tensorflow keras scikit-learn
```

---

### ▶️ Run the Project

1. Clone the repository:
```bash
git clone https://github.com/<your-username>/Deep_learning_training_design_space.git
```

2. Navigate to the folder:
```bash
cd Deep_learning_training_design_space
```

3. Open any `.ipynb` file using:
- Jupyter Notebook, or
- Google Colab
